# Carpeta hardware/

Colocar acá:

- `esquematico.png` (o `.pdf`) — captura completa del esquemático de Proteus.
- `bom.md` o `bom.xlsx` — lista de componentes (Bill of Materials): PIC16F887, cristal 4MHz, 2x capacitor 22pF, sensor TCS3200/TCS230, LCD 16x2, servomotor SG90, potenciómetro 10kΩ, pulsador, resistencias de pull-up/pull-down si aplica, adaptador USB-Serie para UART/AN1310.

Recordatorio: el esquemático en Proteus muestra los capacitores del cristal como "2.2uF" — esto es un error de tipeo del simulador. Los capacitores físicos reales a colocar son de **22pF**, el valor estándar para un cristal de 4MHz.
